# Руководство пользователя (Демонстрационный сценарий)

Это руководство содержит пошаговый сценарий для записи демонстрационного видеоролика и проверки основных функций системы.

## Сценарий 1: Управление пользователями (Администратор)
1. **Вход**: Зайдите в систему под логином `admin` и паролем `admin123`.
2. **Список пользователей**: Перейдите в раздел "Пользователи". Вы увидите таблицу зарегистрированных аккаунтов.
3. **Блокировка**: Нажмите "Заблокировать" напротив любого пользователя. Статус сменится на красный badge "Заблокирован". Нажмите "Активировать", чтобы вернуть доступ.
4. **Смена ролей**: В выпадающем списке напротив пользователя выберите `Admin`. Теперь этот пользователь тоже может управлять системой.

## Сценарий 2: Запись и распознавание (Оператор)
1. **Регистрация**: Выйдите из системы и нажмите "Регистрация". Создайте нового пользователя (например, `operator1`).
2. **Вход**: Войдите под новым аккаунтом.
3. **Запись команды**:
   - Нажмите на красную кнопку с микрофоном.
   - Четко скажите фразу: **"Зарегистрировать трубу номер P45345ИВ"**.
   - Нажмите "Остановить".
4. **Проверка результата**:
   - Система отобразит карточку с текстом, командой "Зарегистрировать" и идентификатором "P45345ИВ".
   - Прослушайте запись через встроенный плеер.
5. **Подтверждение**: Нажмите "Подтвердить". Результат сохранится в базе данных.

## Сценарий 3: Ручная корректировка
1. Сделайте новую запись, например: **"Начать обработку 21957898"**.
2. Если система распознала текст неверно (например, не нашла идентификатор), нажмите кнопку **"Редактировать"**.
3. В появившемся окне вручную исправьте идентификатор или текст.
4. Нажмите "Сохранить изменения".

## Сценарий 4: История и фильтрация
1. Перейдите в раздел "История".
2. Вы увидите список всех ваших и чужих записей.
3. Раскройте блок "Фильтры".
4. Выберите команду `Зарегистрировать` и нажмите "Применить". Список обновится.
5. Введите в поле "Идентификатор" часть кода (например, `P453`) и снова нажмите "Применить".
