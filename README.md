# Система распознавания голосовых команд (MVP)

Данный проект представляет собой веб-приложение для автоматической транскрибации и парсинга голосовых команд операторов. Система использует библиотеку **VOSK** для офлайн-распознавания речи и **Flask** в качестве бэкенд-платформы.

## Основной стек технологий
- **Backend**: Python 3.10+, Flask, Flask-SQLAlchemy (SQLite), Flask-Login.
- **Frontend**: Jinja2, Bootstrap 5, Vanilla JS (MediaRecorder API).
- **Speech-to-Text**: VOSK (Russian model).
- **Audio Processing**: Pydub (требует системный FFmpeg).

---

## Системные требования

### 1. Установка FFmpeg (Критически важно)
Для конвертации аудио из браузерного формата (WebM/Ogg) в формат WAV, необходимый для VOSK, в системе **обязательно** должен быть установлен FFmpeg.

- **Ubuntu/Debian**:
  ```bash
  sudo apt update && sudo apt install ffmpeg
  ```
- **Windows**:
  1. Скачайте бинарные файлы с [gyan.dev](https://www.gyan.dev/ffmpeg/builds/).
  2. Распакуйте и добавьте путь к папке `bin` в системную переменную `PATH`.
- **macOS (Homebrew)**:
  ```bash
  brew install ffmpeg
  ```

## Инструкция по установке и запуску

### 1. Клонирование и окружение
```bash
# Перейдите в директорию проекта
cd ya-profi

# Создайте и активируйте виртуальное окружение
python3 -m venv venv
source venv/bin/activate  # Для Windows: venv\Scripts\activate

# Установите зависимости
pip install -r requirements.txt
```

### 2. Настройка модели VOSK
Для работы распознавания необходимо скачать языковую модель:
1. Перейдите на [alphacephei.com/vosk/models](https://alphacephei.com/vosk/models).
2. Скачайте легковесную русскую модель: **`vosk-model-small-ru-0.22`** (ок. 45 МБ).
3. Распакуйте архив в корень проекта.
4. Переименуйте папку модели в **`model`** (так, чтобы путь был `ya-profi/model/`).

### 3. Запуск приложения
```bash
python main.py
```
Приложение будет доступно по адресу: `http://127.0.0.1:5000`

## Функциональные возможности MVP
1. **Регистрация и Вход**: Операторы могут регистрироваться самостоятельно. Администратор может управлять списком пользователей.
2. **Запись голоса**: Захват звука с микрофона прямо в браузере.
3. **Авто-распознавание**: Извлечение текста, одной из 5 команд и идентификатора (8 цифр или буквенно-цифровой код).
4. **Корректировка**: Оператор может вручную исправить результат, если распознавание было неточным.
5. **История**: Просмотр всех записей с фильтрацией по командам и ID, возможность прослушать аудио.
